
# 🪄 The Wizard's Ledger

A **premium resume builder** inspired by the magical world of **🏰 Hogwarts Legacy**.
Craft your professional identity like a wizard crafting spells — elegant, powerful, and immersive.

Built with **Next.js (App Router)** ⚡, **React** ⚛️, and **Tailwind CSS** 🎨

---

## 🏰 UI Inspiration

🪄 **Hogwarts Legacy**

This app captures the essence of a magical interface:

* 📜 Scroll-like layouts and parchment aesthetics
* ✨ Subtle glow, depth, and mystical accents
* 🧙 Elegant typography and structured sections
* 🪄 Spellbook-style interactions

The goal: transform resume building into a **magical, immersive experience**.

---

## ✨ Features

📄 **Dynamic Resume Builder**

* Sections:

  * 🎓 Education
  * 💼 Experience
  * 🧪 Projects
* Add / remove entries dynamically

🧠 **Live Preview System**

* Real-time updates as you type
* Split layout:

  * 📝 Form (left)
  * 📜 Resume preview (right)

✨ **Smart Preview Logic**

* Shows **rich demo resume** only when form is empty
* Instantly switches to user-only data on input
* ❌ No mixed placeholder + real data

💾 **Persistence**

* Saves resume data in localStorage
* Auto-loads on revisit

🧹 **Reset Controls**

* Clear form + stored data instantly

📥 **PDF Export**

* Download resume as PDF
* Powered by `html2pdf.js`
* Clean, print-ready output

🎬 **Smooth UI Experience**

* Subtle animations
* Polished transitions
* Clean, readable design
