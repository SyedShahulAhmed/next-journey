# ⚔️ Habit Tracker — Codex of Discipline

A habit tracking application built with **Next.js + TypeScript + Tailwind CSS**, designed as a **Norse-inspired discipline codex**.

This is not just a tracker — it represents a system where each habit is an **oath**, each completion is a **mark in time**, and consistency builds a **streak of discipline**.

---

## ✨ Features

* ➕ Add, edit, and remove habits
* ✅ Daily completion tracking
* 🔥 Streak system (continuous days tracking)
* 📊 Progress and completion stats
* 🔍 Filter (All / Completed / Pending)
* 💾 Local storage persistence
* 📅 Streak heatmap (timeline view)

---

## 🎨 Theme

Inspired by **God of War (Norse Mythology)**:

* Dark, stone-like UI
* Gold and frost accents
* Minimal, powerful layout
* Designed to feel like a **warrior’s discipline codex**, not a modern app

---

## 🛠️ Tech Stack

* **Next.js (App Router)**
* **TypeScript**
* **Tailwind CSS**
* **LocalStorage**
